# About

[Dartlang](https://www.dartlang.org) wrapper for [flickr API](https://www.flickr.com/services/api/) calls.

# Install

In your `pubspec.yaml`:

```yaml
dependencies:
  flickr:
    git: git@github.com:damondouglas/flickr.dart.git
```
